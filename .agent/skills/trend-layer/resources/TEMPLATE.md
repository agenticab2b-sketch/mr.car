## F) Motion discipline (минимум 6 правил)
- где движение помогает понять изменение состояния

- что запрещено (вечные анимации, резкие прыжки, хаос)
- правило “уменьшить движения” (reduced motion)
- правило скорости/умеренности (без цифр, но с ясным смыслом)
- правило для переходов между блоками/экранами (аккуратно, без шоу)

## G) Glass / Liquid-glass (если выбран) — безопасные ограничения (минимум 7)
Пункты “делай так / не делай так”:
- стекло только на крупных контейнерах (1–2 места)
- текст всегда читаемый (контраст/подложка)
- фон под стеклом не должен быть слишком шумным
- не наслаивать стекло в 3–4 слоя
- не делать стекло на мелких контролах, где важна ясность клика
- в dark mode проверять отдельно
- на mobile — осторожнее (может быть упрощение)

Если стекло не выбрано — напиши: NOT USED
