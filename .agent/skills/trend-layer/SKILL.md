# Trend Layer (WOW) — Optional Modern Visual Effects for Web

## Цель
Добавить к сайту **современный “вау-слой”** (трендовые визуальные приемы и микро-движения),
но так, чтобы:
- это было **уместно** для проекта и бренда,
- не страдала **читаемость**, доступность и восприятие,
- эффект был **точечным**, а не “везде всё блестит и движется”.

Trend Layer — это **не замена** базовых систем (Typography/Color/Layout/Shape/Components/Content/Motion),
а **надстройка** поверх них.

## Когда использовать
Используй этот skill, когда нужно:
- добавить “wow” в hero-секцию, ключевые карточки, CTA, промо-блоки;
- сделать сайт визуально “в тренде” (в пределах здравого смысла);
- предложить 2–3 варианта “эффектных” приемов и аккуратный план, как не переборщить;
- сделать эффект **опциональным** и легко вырезаемым, если мешает.

## Когда НЕ использовать
Не используй Trend Layer, если:
- сайт должен быть максимально строгим/консервативным (юридические услуги, гос. тематика) и заказчик явно против “вау”;
- контент уже сложный и перегруженный (сначала исправь структуру/иерархию);
- нет ясного ответа “зачем нужен эффект” (просто “чтобы было красиво” — слабая причина).

## Входные данные (минимум)
Если чего-то нет — сделай **не более 1–2 допущений** и явно укажи строкой `Assumption: ...`.
- тип проекта (автосервис / B2B / магазин / личный бренд)
- тон бренда: строго / дружелюбно / премиально / технично / “индастриал”
- где нужен WOW: hero / CTA / карточки услуг / прайс / галерея работ / отзывы
- ограничения: “не раздражает”, “не отвлекает”, “без тяжёлых эффектов”, “нужен dark mode” (если есть)
- устройства: desktop + tablet + mobile (обязательно учитывать)

## Жёсткие правила
1) Trend Layer всегда **точечный**: максимум 1–2 “герой-приема” на страницу.
2) WOW не должен ухудшать **контраст**, **читаемость**, **понятность кликов**.
3) Любой эффект должен иметь **fallback-версию** без эффекта (по смыслу и структуре — то же самое).
4) Любое движение/анимация: “есть смысл → есть движение”. Если смысла нет — убираем.
5) Никаких “вечных” движений по умолчанию. Только короткие реакции/переходы, либо очень умеренные фоновые эффекты.
6) Структуру ответа из `resources/TEMPLATE.md` **нельзя сокращать или переставлять**.

## Decision tree (как выбрать тренд-пак)
- Если проект “премиально/технологично” → Glassmorphism (очень дозированно) + мягкие microinteractions.
- Если проект “индастриал/брутально” → Neubrutal accents (толще контуры, контраст, графика) + короткие реактивные эффекты.
- Если проект “много сервисов/много блоков” → Bento-style композиция (карточки разного размера) + минимальный акцент на 1 ключевом блоке.
- Если аудитория широкая и важна понятность → microinteractions + умеренный expressive motion, без “эффектов ради эффектов”.
- Если заказчик сомневается → предложи 2 варианта: “Safe WOW” и “Bold WOW”, с простым планом отката.
